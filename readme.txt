A simple form to create url for tracking of custom campaigns on Google Analytics

== Description ==

Thanks to Google Analytics you can tracking your online campaign with different tags and source information. For make it, Google Analytics usa some parameters that you must attach to the urls that you want use on yours campaigns
This plugin allows you to successfully add these parameters to the URL to be used in custom advertising campaigns on web.

For more information visit: [lucaraldiri.it](http://www.lucaraldiri.it).

== Installation ==

1. Upload `tracking-url-builder-for-analytics` directory to the `/wp-content/plugins/` directory or use plugin search - Admin > Plugins > Add new > Search for 'Tracking URL Builder for Analytics'.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Tag and copy your URLs.

== Frequently Asked Questions ==

= It's necessary to have a Google Analytics Account to use this plugin? =

It is not strictly necessary to have a Google Analytics account to use this plugin, but if we want to measure the effects of our online custom campaigns we will have to have an account Google Analytics.

= I need to modify my Google Analytics tracking code? =

No, if you have properly set your tracking code you will not have to do anything else.

= I have activate this plugin, where is Builder form? =

You can find Tracking URL Builder for Analytics form under tools voice of menu in your WordPress administration

= Can I have support on this plugin? =

Use this plugin is very simple but, if you have any questions, doubts or suggestions, [you can submit me your requests on my site](http://www.lucaraldiri.it/contatti/). I usually reply in 24/48 hours.
